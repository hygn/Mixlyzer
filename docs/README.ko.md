# Mixlyzer

언어: [English](../README.md) | [日本語](README.ja.md)

## 소개

Mixlyzer는 DJ와 프로듀서를 위한 트랙 분석·편집 도구입니다. 곡 전체를 스캔해 BPM/위상 변화를 잡아내고, 키 변조를 추적하며, JumpCUE 후보를 제안합니다. 인터랙티브한 편집기로 결과를 다듬은 뒤 외부 플랫폼으로 내보낼 수 있습니다.

## 주요 기능

- **정적 비트그리드 분석** - 곡의 전체 BPM과 위상을 추정해 시작 그리드를 만듭니다.  
- **동적 비트그리드 분석** - 곡을 여러 세그먼트로 나눠 BPM·위상 변화를 정확히 매핑합니다.  
- **동적 키 분석** - 시간에 따른 키 변조를 추적해 믹스 시 음정 일관성을 유지합니다.  
- **JumpCUE 분석** - 전/후진 점프에 적합한 큐 페어를 찾아 창의적인 점프를 지원합니다.  
- **빠른 인터랙티브 편집** - 세그먼트 기반 비트그리드 도구, 부분 재분석, Undo/Redo로 빠르게 작업합니다.  
- **내보내기** - 보정된 템포 세그먼트, 다운비트, JumpCUE를 포함한 Rekordbox XML 내보내기.

## 이런 분께 추천합니다

- 템포 변동이 있는 곡을 안정적으로 플레이하고 싶은 DJ  
- 애니송/동인/게임·보컬로이드·VTuber 등 템포·키 변동이 잦은 씬을 다루는 DJ  
- 트랙을 재구성하기 전에 믿을 만한 분석 데이터를 원하는 프로듀서/리믹서

## 시작하기

### 방법 A: uv 사용 (개발용 권장)
1. [uv](https://github.com/astral-sh/uv)를 설치합니다.  
2. 저장소를 클론하고 프로젝트 폴더로 이동합니다.  
3. `uv sync`로 의존성과 가상환경을 준비합니다(Python 3.12+).  
4. `uv run python -m app.main`으로 Mixlyzer를 실행합니다.

### 방법 B: 배포 바이너리 사용
1. Releases 페이지에서 최신 빌드를 받습니다.  
2. 원하는 위치에 압축 해제합니다.  
3. 실행 파일을 시작합니다(예: Windows는 `Mixlyzer.exe`).  
4. 최초 실행 시 라이브러리 경로를 지정하고 바로 분석·편집·내보내기를 시작하세요.  

## Typical Workflow

1. **트랙 로드** - 파일을 드래그하거나 라이브러리에서 선택합니다.  
2. **자동 분석** - 템포 세그먼트, 키, JumpCUE가 자동 검출됩니다.  
3. **그리드 다듬기** - Beatgrid 편집기로 BPM 조정, 세그먼트 이동, 부분 재분석을 수행합니다.  
4. **내보내기** - 편집 결과를 저장하고 Rekordbox XML로 내보내어 부스까지 가져갑니다.
